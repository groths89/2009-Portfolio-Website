DOLPHIN PROJECT

This image was produced entirely with Illustrator. The basic idea for this assignment was to create a sign dealing with a dolphin. This was part of a group of projects the airplane, a blimp, and a person. You had to create four signs for each of the items. The dolphin you see above started out as a real photograph, then I traced it with the pen tool. I decided to use green for the dolphin because you don't usually see a green dolphin in the waters, so it would be noticeably different from the original picture.
